# it-handbook.py

Личный справочник и база знаний по алгоритмам и структурам данных (DSA) с примерами реализации на Python + смежные материалы.

## 📌 О проекте
Собираю теорию, решаю задачи и систематизирую знания, полученные в процессе обучения. 

**Состав материалов:**
*   **LeetCode Patterns:** Разбор типовых задач и паттернов (Two Pointers, Sliding Window, BFS/DFS и др.).
*   **Поколение Python:** Некоторые упражнения и задачи из курсов экосистемы Stepik.
*   **Personal Notes:** Мои конспекты по теории Computer Science и особенностям реализации структур данных на Python.

## 📂 Структура репозитория
* `/dsa-handbook` — конспекты, шпаргалки и описание алгоритмов и строуктур данных
* `/generation-python` — материалы из курсов «Поколение Python»
* `/leetcode` — задачи с leetcode
* `/syntax-py` - шпаргалки по синтаксису Python
